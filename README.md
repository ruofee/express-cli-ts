# 🍁 express-cli-ts

## 🚗 介绍

一款提升工程效能的Nodejs脚手架, 快速生成typescript + express项目

## 💻 如何使用

全局安装`express-cli-ts`:

```bash
# use yarn
yarn global add express-cli-ts
# use npm
npm i express-cli-ts --global
```

创建项目:

```bash
express-cli init <your_project>
# or
express-cli-ts init <your_project>
```

## 📦 脚手架集成

- typescript
- 网络库(axios)
- 单元测试(mocha + chai)
- 路由
